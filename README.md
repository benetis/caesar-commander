![Screenshot](docs/screenshot.png)

### Shortcuts

•	↑ / ↓ — Move up/down through items
•	← / → or Tab — Switch between left/right panes
•	Enter — Open selected folder
•	Backspace — Go up one directory level
•	F6 — Move selected item to the other pane
